# Myraid Box (万象盒子)
包罗万象，从一个集成开始。每日一言、每日诗词、每日天气、每日油价......还有更多的每日

![hacs_badge](https://img.shields.io/badge/HACS-Custom-41BDF5.svg)

多服务数据集成平台，支持：
- 每日一言
- 经典诗词
- 天气预报
- 油价查询

## 安装
1. 通过 HACS 添加自定义仓库
2. 搜索 "Myraid Box"
3. 点击安装

## 配置
通过 Home Assistant 的 `配置` > `集成` 添加。

[完整文档](https://github.com/938134/myraid_box)
