# 万象盒子 (Myraid Box) - 智能生活数据聚合平台

![HACS Custom](https://img.shields.io/badge/HACS-Custom-41BDF5.svg)
![Version](https://img.shields.io/badge/version-2.0.0-blue.svg)
![Home Assistant](https://img.shields.io/badge/Home%20Assistant-2023.8%2B-orange.svg)

**包罗万象，从一个集成开始**

万象盒子是一个专为 Home Assistant 设计的智能数据聚合平台，集成了多种实用的日常数据服务，让您的智能家居更加丰富多彩。

## 🚀 核心特色

### 📝 每日一言
- **数据源**: 一言官网 (Hitokoto)
- **功能**: 每日励志名言，支持多种分类
- **传感器**:
  - 名言内容
  - 名言分类（文学、影视、哲学等）
  - 名言作者
  - 名言来源

### 📚 每日诗词  
- **数据源**: 今日诗词 API
- **功能**: 经典古诗词欣赏，陶冶情操
- **传感器**:
  - 精选诗句
  - 诗词标题
  - 诗人信息
  - 诗词朝代
  - 完整诗词
  - 诗词译文

### 🌤️ 每日天气
- **数据源**: 和风天气 (官方 JWT 认证)
- **功能**: 3天精准天气预报
- **特色**: 使用 EdDSA 算法安全认证
- **传感器**:
  - 今日天气详情
  - 温度、湿度、风力
  - 降水量、气压
  - 紫外线指数
  - 未来两天预报

### ⛽ 每日油价
- **数据源**: 汽油价格网
- **功能**: 全国各省市实时油价
- **传感器**:
  - 92号/95号/98号汽油价格
  - 0号柴油价格
  - 省份信息
  - 调价窗口期
  - 油价走势预测

### 📜 每日历史
- **数据源**: 今天历史网
- **功能**: 了解历史上的今天
- **传感器**:
  - 历史事件内容
  - 事件发生年份
  - 历史时期分类
  - 事件详情列表

## 🔧 安装指南

### 通过 HACS 安装（推荐）
1. 打开 HACS
2. 点击「集成」
3. 点击右下角「+ 浏览并下载仓库」
4. 搜索「万象盒子」或「Myraid Box」
5. 点击「下载」
6. 重启 Home Assistant

### 手动安装
1. 下载最新版本
2. 将 `custom_components/myraid_box` 文件夹复制到您的 Home Assistant 配置目录
3. 重启 Home Assistant

## ⚙️ 配置说明

### 初始配置
1. 进入 Home Assistant 「配置」→「设备与服务」
2. 点击「+ 添加集成」
3. 搜索「万象盒子」
4. 选择要启用的服务
5. 按步骤配置每个服务

### 服务配置详情

#### 每日一言
- **更新间隔**: 10分钟（默认）
- **分类选择**: 动画、漫画、游戏、文学、原创、网络、影视、诗词、哲学、随机

#### 每日诗词  
- **更新间隔**: 10分钟（默认）
- **API**: 自动 Token 管理，无需配置

#### 每日天气 ⭐
- **更新间隔**: 30分钟（默认）
- **认证方式**: JWT + EdDSA 私钥
- **配置要求**:
  1. 注册和风天气开发者账号：[https://dev.qweather.com/](https://dev.qweather.com/)
  2. 创建项目获取项目ID、密钥ID和EdDSA私钥
  3. 私钥字段已加密显示，确保安全输入

#### 每日油价
- **更新间隔**: 360分钟（默认）
- **支持省份**: 全国31个省市自治区

#### 每日历史
- **更新间隔**: 10分钟（默认）

## 🔒 安全特性

- **密码字段加密**: 所有敏感信息（如私钥）在配置界面中均显示为密码输入框
- **Token 自动管理**: 诗词服务自动处理 API Token 的获取和刷新
- **JWT 认证**: 天气服务使用行业标准的 JWT 认证
- **数据本地缓存**: 减少不必要的 API 调用
